# Setup Instructions

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed
- Have [npm](https://www.npmjs.com/) available for package management

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/prolevans1/haunt.git
   ```
2. Navigate to the project directory:
   ```bash
   cd haunt
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application
To start the application, run:
```bash
npm start
```
